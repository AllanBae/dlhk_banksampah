## 📖 Deskripsi

**DLHK Bank Sampah** adalah aplikasi berbasis web yang dirancang untuk membantu pengelolaan sistem bank sampah secara digital.
Aplikasi ini mempermudah pencatatan transaksi, pengelolaan nasabah, serta monitoring saldo dan laporan secara efisien.

---

## ✨ Fitur Unggulan

* 👤 **Manajemen Nasabah**
* ♻️ **Setoran Sampah** (otomatis hitung nilai)
* 💰 **Penarikan Saldo**
* 📊 **Laporan & Rekap Data**
* 🔎 **Pencarian & Filter Data**
* 🧾 **Riwayat Transaksi**

## ⚙️ Instalasi

### 1. Clone Repository

```bash
git clone https://github.com/AllanBae/dlhk_banksampah.git
```

### 2. Masuk ke Folder

```bash
cd dlhk_banksampah
```

### 3. Pindahkan ke Laragon

```
C:\laragon\www\
```

### 4. Import Database

* Buka phpMyAdmin
* Import file `.sql`

### 5. Jalankan Project

```
http://localhost/dlhk_banksampah
```
