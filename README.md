# dlhk_banksampah

Sistem informasi Bank Sampah DLHK berbasis web yang digunakan untuk mengelola data nasabah, transaksi setoran sampah, penarikan saldo, serta laporan

Fitur Utama
👤 Manajemen Data Nasabah
♻️ Pencatatan Setoran Sampah
💰 Penarikan Saldo
📊 Laporan Transaksi
🔍 Pencarian dan Filter Data

Teknologi yang Digunakan
PHP (Native / CodeIgniter)
MySQL Database
HTML, CSS, JavaScript
Laragon (Local Server)
